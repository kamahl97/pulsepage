# Launch Day Checklist

Use before launching a product, ad, website, sale, service, event, or campaign.

## Before posting the link

- Homepage loads
- Offer page loads
- Checkout/booking/contact works
- Confirmation email/message arrives
- Price is correct
- Coupon code works, if used
- Mobile layout is usable
- Main image/video loads
- Terms/refund/contact info are visible
- Analytics/pixel installed, if used
- Backup contact method ready
- Monitor active on main page and money page

## After posting

- Click your own link from the live post/email/profile.
- Test on phone using cellular data.
- Check for first visitor/order/form submission.
- Watch error messages for the first hour.

## If something breaks

Pause traffic, switch to backup link, post a short fix message, and log the cause.
