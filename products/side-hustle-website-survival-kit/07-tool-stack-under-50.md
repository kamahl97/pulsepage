# Tool Stack Under $50

A low-budget stack for keeping a small money site alive.

## Free / low-cost basics

- Domain registrar account with auto-renew enabled
- Hosting account with payment method current
- Free SSL certificate from host/Let’s Encrypt
- Google Search Console
- Google Analytics or privacy-friendly analytics
- Uptime monitor such as PulsePage or another simple monitor
- A backup contact form/link
- A spreadsheet for weekly checks

## Optional paid spend

- $9-$19: domain privacy/renewal buffer if needed
- $0-$20: simple logo/banner if needed
- $0-$10: Gumroad/Etsy listing images made with free tools
- $0-$50: only spend on traffic after checkout/contact is verified

## Don’t spend first on

- Fancy logo packages
- Generic “business courses”
- Paid ads before the site path works
- Expensive tools before you have sales
