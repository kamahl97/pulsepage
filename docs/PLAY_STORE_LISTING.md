# PulsePage Google Play Store Listing Draft

## Recommended app name
PulsePage: Website Monitor

Reason: clear, searchable, and easier to understand than HustleGuard. We can still use HustleGuard as a tagline later.

## Short description
Simple website and server uptime monitoring for side hustlers.

## Full description
Your website is part of your money machine. PulsePage helps you know when it goes down.

PulsePage is a simple uptime monitor built for side hustlers, freelancers, indie builders, and small business owners who need a fast way to watch important websites and server ports.

Use PulsePage to monitor:
- websites and landing pages
- client sites
- small business pages
- VPS/server ports
- SSL certificate expiration
- response time and recent uptime history

Core features:
- Add website monitors with normal URLs like example.com
- Add server port monitors like 31.220.56.145:22
- See whether each monitor is up or down
- Check response time
- View recent check history
- Track SSL certificate days remaining for HTTPS websites
- Get alerts when a monitor changes status

PulsePage is intentionally simple. It is not built for giant enterprise teams. It is built for people who run small online businesses and want peace of mind without complicated dashboards.

Free plan idea:
- Monitor up to 3 websites or server ports
- Standard check intervals
- Basic history

Pro plan idea:
- More monitors
- Faster checks
- Longer history
- More alert options

PulsePage does not request unnecessary permissions. It does not need your contacts, photos, location, SMS, microphone, or camera.

Know when your money page goes down before customers do.
