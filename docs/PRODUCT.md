# PulsePage Product Plan

## Target customer
People who run small online things and cannot afford to miss downtime:
- side hustlers
- freelancers
- small business owners
- indie app builders
- Shopify/landing page owners
- VPS beginners

## Core promise
"Know when your money page goes down before customers do."

## MVP features
1. Add website URL or server port target.
2. Check website HTTP status and response time.
3. Check TCP server ports like SSH, game servers, or app ports.
4. Show current status: up, down, slow, warning.
5. Show recent check history.
6. Show SSL certificate expiry for HTTPS URLs.
7. Later: push/email alerts.

## Pricing idea
- Free: 3 monitors, 15-minute checks, 24-hour history.
- Pro: $2.99/month or $24.99/year.
- Later business tier: $7.99/month.

## First app store positioning
Title ideas:
- PulsePage: Website Monitor
- HustleGuard Uptime Monitor
- SitePulse: Uptime Alerts

Short description:
Simple website and server uptime monitoring for side hustlers and small businesses.
